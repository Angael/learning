# Resources

This file tracks sources for teaching and project decisions.

## Teaching and learning

- Matt Pocock, "Learn Anything With My /teach Skill" — https://www.aihero.dev/learn-anything-with-my-teach-skill
  - Why: explains the stateful teaching-workspace model used here.
- Matt Pocock `teach` skill source — https://github.com/mattpocock/skills/tree/main/skills/productivity/teach
  - Why: canonical skill structure for missions, resources, sessions, references, and learning records.
- Barak Rosenshine, "Principles of Instruction" — https://www.aft.org/sites/default/files/Rosenshine.pdf
  - Why: concise evidence-informed principles: review, small steps, questioning, models, guided practice, checks for understanding, scaffolds, independent practice, and review.
- Cognitive apprenticeship overview — https://learning-theories.com/cognitive-apprenticeship-collins-et-al.html
  - Why: useful model for making expert thinking visible through modeling, coaching, scaffolding, articulation, reflection, and exploration.
