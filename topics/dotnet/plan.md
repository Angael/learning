# Topic plan: .NET

## End goal

Angeal can build and reason about practical .NET backend services, from C# fundamentals to deployment-aware APIs.

## Progression

1. C# syntax and runtime mental model.
2. Projects, packages, and `dotnet` CLI.
3. ASP.NET Core request pipeline.
4. Dependency injection and configuration.
5. Data access and validation.
6. Testing, logging, containers, deployment.

## Achievements

- [ ] Explain what the CLR/runtime does.
- [ ] Build a minimal API endpoint.
- [ ] Add configuration and DI.
- [ ] Write a test for endpoint behavior.
