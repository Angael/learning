# Topic plan: SSO

## End goal

Angeal can reason about SSO systems, choose between OAuth/OIDC/SAML patterns, and debug common production auth failures.

## Progression

1. Identity vs authentication vs authorization.
2. Sessions, cookies, tokens, and redirects.
3. OAuth roles and authorization code flow.
4. OIDC identity layer and ID tokens.
5. SAML mental model and enterprise SSO.
6. Threat models, debugging, and production operations.

## Achievements

- [ ] Draw the authorization code flow from memory.
- [ ] Explain access token vs ID token.
- [ ] Diagnose a redirect URI mismatch.
- [ ] Compare OIDC and SAML tradeoffs.
