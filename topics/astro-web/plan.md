# Topic plan: Astro web

## End goal

Angeal can build, modify, debug, and explain this Astro site from first principles.

## Progression

1. Static page mental model: source files become HTML/CSS/JS output.
2. Astro components and frontmatter.
3. Routing and content organization.
4. Styling and accessibility basics.
5. Content collections for lessons/topics.
6. Build output and deployment artifacts.

- [ ] Explain what `npm run build` produces.
- [ ] Add a new page without help.
- [ ] Create a reusable component.
- [ ] Convert a lesson into structured content.
- [ ] Debug a broken build.
