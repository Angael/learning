# Notes: value types, reference types, and null

## Lesson summary

Published on 2026-07-05. This lesson teaches the practical assignment model for C#: value types copy the value; reference types copy a reference to the same object. It introduces `null`, `NullReferenceException`, and the modern nullable-reference-type distinction between `string` and `string?`.
