## Why this lesson
First lesson in the Astro web track. No prior completion evidence exists, so this introduces the smallest useful mental model: source files become browser-ready build output.

## Concept taught
Astro processes files in `src/`, uses files in `src/pages/` as routes, and emits static output under `dist/` via `npm run build`. `public/` assets are copied rather than transformed.
